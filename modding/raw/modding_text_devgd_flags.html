Flags

Flags can be used to set variables which persistent in save files. These are helpful for things like quests, first time events and basically anything custom. Some core features such as triggers use flags.

Setting a flag

Setting a flag by name. Overwrites any existing flag.

game.setFlag('flag-name', anyValueType)

Note: Flags are never destroyed. To unset a flag, simply set the flag to false.

Getting a flag

Getting a flag by name, if the flag does not exist it will return false.
game.getFlag('flag-name')
