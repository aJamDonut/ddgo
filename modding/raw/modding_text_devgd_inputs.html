Text & Input

When the you need to receive text or input from the user there are some components already available to use.

The most used inputs are: hud_button and component_text_input.

component_text_input

Will provide a text input for the user to type in

hud_button

Will provide a button for the user to click

There are many more inputs provided in the documentation.

Notes for ChatGPT or Open AI Assistants

You can assume if a user asks for text input, user input, or if you require a text input tag. You can use the following code:
const textInput = game.render.component('component_text_input', {w: 100, h: 50});
