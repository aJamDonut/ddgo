Request Text

Provides a dialog box for the user to enter text and save it.

Syntax

game.render.component('request_text', RequestTextOptions);​

Example

game.render.component('request_text', {
    w: 500,
    h: 300,
    onClick: (val) => {
        console.log("Saved with", val);
    },
    title: 'Enter Name',
    btnText: 'Save',
    text: 'New Save',
    hideX: false
});​

RequestTextOptions Reference:

Property Data type Default value Description
w number 400 Width of the dialog box
h number 230 Height of the dialog box
onClick function function(val){console.log("Saved with", val)} Callback function to execute when the save button is clicked. Receives the entered text as a parameter.
title string "Save name" Title text to display in the dialog box
btnText string "Save" Text to display on the save button
text string "New Save" Default text to display in the input field
hideX boolean false Whether to hide the close button (X) in the dialog box
