Slider

Provides a draggable slider component that allows users to select a value within a specified range.

Syntax

game.render.component('hud_slider', HudSliderOptions);​

Example

game.render.component('hud_slider', {
    name: 'Volume',
    w: 300,
    h: 10,
    min: 0,
    max: 100,
    value: 50
});​

HudSliderOptions Reference:

Property Data type Default value Description
w number 300 Width of the slider
h number 10 Height of the slider
min number 1 Minimum value of the slider
max number 100 Maximum value of the slider
value number 0 Initial value of the slider
