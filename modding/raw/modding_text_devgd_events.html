Events

Events are how the system communicates between complex systems.

If you want to trigger an event when something happens, events are what you want.

You can register listeners for existing events and create your own events.

Syntax

Listen for an event

game.ee.on(‘event-name’, (passThru)=>{
 //...Your code here
});
Important: Don't forget, in javascript objects are always passed by reference. If you modify the object being passed it will have side effects

Trigger a new event

game.ee.emit(‘event-name’, passthru);
