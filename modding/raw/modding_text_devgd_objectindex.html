Object Index

The Object Index can be used to store and find objects in the game. This is useful for performance and grouping objects.

Syntax

Find a specific instance given its ID

game.index.find(objectId);

Get all instances with a given codename

game.index.getIndex("IndexName");

Add item to an index

Note: If the index does not exist, it will create it.

If an optional key is provided then this key is used as the index for this item as opposed to the objects id.

game.index.addToIndex("IndexName", objectToAdd, optionalKey);
