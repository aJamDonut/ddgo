Moving and Tweens

UI Components, Sprites and Items can be moved on screen with Tweens. This is different from moving with pathfinding. This should only be used for moving UI Component or in game animations.

All tweening is implemented uses CreateJS TweenJS. Any functionality in TweenJS exists here.

Syntax

const tween = createjs.TWEEN.get({ObjectToTween}, {TweenOptions}).to({TweenToObject}, {TweenDuration}, {Easing})

Example

Slide an item from one side to the other

const tween = createjs.Tween.get(obj, {override: false, loop: false}).to(
 {
  x: opts.x,
  y: opts.y
 },
 opts.duration || 100,
 createjs.Ease.quintOut
);

Notes for ChatGPT or Open AI Assistants

You can assume that TWEEN.TWEEN in our system is called createjs.TWEEN and contains exactly the same functionality as TWEEN.TWEEN.

If the user ever asks to "shake" "move", "slide" or otherwise change the position of an object, always use a tween function.
