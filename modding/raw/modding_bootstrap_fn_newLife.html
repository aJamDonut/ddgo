New Life

Create a new life object with specified properties.

Syntax

game.util.newLife(isPlayer, level, species, faction, bodies, brains, masks, weapons, backWeapons, dummyObject)

Example

game.util.newLife(false, 5, "human", "nomad", ["body1", "body2"], ["brain1", "brain2"], ["mask1", "mask2"], ["weapon1", "weapon2"], ["backWeapon1", "backWeapon2"], false);

New Life Reference:

Property Data type Default value Description
isPlayer Boolean false Specifies if the life object is a player
level Number 0 The level of the life object
species String "none" The species of the life object
faction String "nomad" The faction of the life object
bodies Array [] An array of body options for the life object, one will be randomly chosen
brains Array [] An array of brain options for the life object, one will be randomly chosen
masks Array [] An array of mask options for the life object, one will be randomly chosen
weapons Array [] An array of weapon options for the life object, one will be randomly chosen
backWeapons Array [] An array of back weapon options for the life object, one will be randomly chosen
dummyObject Boolean false Specifies if the life object is a dummy object
