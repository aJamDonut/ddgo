Scrolling Text

Provides a scrolling text effect where the text is revealed letter by letter.

Syntax

game.render.component('component_scrolling_text', ScrollingTextOptions);​

Example

game.render.component('component_scrolling_text', {
    text: "I am talking...",
    w: 300,
    h: 50,
    delay: 100
});​

ScrollingTextOptions Reference:

Property Data type Default value Description
text string "I am talkin..." The text to be displayed
w number 300 Width of the scrolling text container
h number 50 Height of the scrolling text container
delay number 100 Delay in milliseconds for each letter to be revealed
