Scrollbox

Provides a scrollable container for displaying content.

Syntax

game.render.component('component_scrollbox', ScrollboxOptions);​

Example

game.render.component('component_scrollbox', {
    w: 500,
    h: 500
});​

ScrollboxOptions Reference:

Property Data type Default value Description
w number 500 Width of the scrollbox
h number 500 Height of the scrollbox
