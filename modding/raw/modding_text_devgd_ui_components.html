UI Components

The user interface is built up of component which can be rendered with game.render.component('component_name', {options}).

To create a component the component code should be made e.g.

_BLUEPRINTS.COMPONENTS.your_component = {
  options: "comma-seperated options",
  getElement: function (options) {
    if (options == undefined) {
      options = {};
    }
    var pane = new UIContainer();
    
    //... Insert custom UI code here

    return pane;
  },
  parent: "component_composites",
  name: "NPC Chatter Talk",
};

Important: All components should be wrapped in a UIContainer as shown in the example. This is important to allow for extendability later on.
