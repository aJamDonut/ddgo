Third Party Libraries

The engine uses several third party libraries to provide different functionality such as rendering or audio.

HowlerJS

HowerJS, the sound library is available globally using window.Howler

window.Howler.mute(true);

Pixi Viewport

Pixi viewport is already initialized as the world stage, but if you require your own use of pixi-viewport you can create a new instance with the following example:
const viewport = new ABE.Viewport.Viewport({...options});

TweenJS

TweenJS, the tweenings/animation library is also available globall on window.createjs.Tween

For example:

createjs.Tween.removeTweens(tweenObject)

lzstring

lzstring, the library used for compression and decompression is also available globally

For example:

LZString.compressToUTF16(string);
