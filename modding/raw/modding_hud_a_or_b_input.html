Radio Input

Provides a set of radio buttons for selecting between two options.

Syntax

game.render.component('hud_radioinput', HudRadioInputOptions);​

Example

game.render.component('hud_radioinput', {
    w: 200,
    h: 50,
    option1: 'Option 1',
    option2: 'Option 2',
    defaultValue: 0,
    onChange: (selectedOption) => {
        console.log('Selected option:', selectedOption);
    }
});​

HudRadioInputOptions Reference:

Property Data type Default value Description
w number 150 Width of the radio input
h number 40 Height of the radio input
option1 string "op1" Text for the first option
option2 string "op2" Text for the second option
defaultValue number 0 Default selected option (0 for option1, 1 for option2)
onChange function null Function to run when an option is selected. Receives the selected option as a parameter.
