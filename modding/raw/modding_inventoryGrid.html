Inventory Grid

Displays an inventory grid which accepts items of a given size xBlocks x yBlocks.

Syntax

game.render.component('inventoryGrid', InventoryGridOptions);​

Example

const inventoryGrid = game.render.component('inventoryGrid', {
    xBlocks: 4,
    yBlocks: 4,
    inventory: playerInventory
});​
inventoryGrid.x = 10;
inventoryGrid.y = 10;
pane.addChild(inventoryGrid);

InventoryGridOptions Reference:

Property Data type Default value Description
xBlocks number 1 Number of blocks in the x-axis
yBlocks number 1 Number of blocks in the y-axis
inventory object undefined The inventory object to display

Notes for ChatGPT or Open AI Assistants:

If the user asks for an inventory or inventory with x slots, you can assume they want to use this component. xBlocks and yBlocks are the equivalent of 2x2 being xBlocks = 2, yBlocks = 2
