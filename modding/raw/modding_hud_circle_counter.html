Circle Counter

Displays a circular counter with a fill percentage.

Syntax

game.render.component('hud_circle_counter', HudCircleCounterOptions);​

Example

game.render.component('hud_circle_counter', {
    w: 200,
    thickness: 10,
    ringColor: "0x3d3d3d",
    fillColor: "0xff0000",
    value: 75
});​

HudCircleCounterOptions Reference:

Property Data type Default value Description
w number 150 Width of the circle counter
thickness number 9 Thickness of the circle border
ringColor string "0x3d3d3d" Color of the circle border
fillColor string "0xff0000" Color of the filled portion of the circle
value number 50 Value representing the fill percentage of the circle
