Bars Counter

Displays a set of bars to represent a value, with customizable options.

Syntax

game.render.component('hud_bars_counter', HudBarsCounterOptions);​

Example

game.render.component('hud_bars_counter', {
    value: 7,
    orientation: 'vertical',
    barWidth: 15,
    barHeight: 30,
    barCount: 8,
    barSpacing: 3,
    barTint: 0x00FF00,
    warning: true
});​

HudBarsCounterOptions Reference:

Property Data type Default value Description
value number 5 The current value to display
orientation string "horizontal" The orientation of the bars ("horizontal" or "vertical")
barWidth number 10 The width of each bar
barHeight number 25 The height of each bar
barCount number 10 The total number of bars to display
barSpacing number 5 The spacing between each bar
barTint number 0x40FF77 The color of the bars
warning boolean false Whether to enable warning colors for low values
