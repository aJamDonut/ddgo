UI Events

Any components (since they use UIContainer) can handle all events as provided in PIXI. For example:

const button = new UIContainer();

button.isInteractive(); //Must always set it to interactive first
button.on('pointerdown', ()=>{
    console.log("I was clicked!")
})

Handling UI updates on tick

To run an update on the UI you can also use the additional onTick function.
pane.onTick(() => {
  //Your code here
});​
For example

pane.onTick(() => {
  someTextThing.text = options.life.data.name; //Update the text every tick, to the be name of the NPC
});​
