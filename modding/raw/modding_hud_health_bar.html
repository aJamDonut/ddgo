Health bar

Displays a health bar with a customizable value, bar color, and text color.

Syntax

game.render.component('hud_health_bar', HudHealthBarOptions);​

Example

game.render.component('hud_health_bar', {
    w: 200,
    h: 30,
    value: 75,
    barColor: "0xFF0000",
    textColor: "0xFFFFFF"
});​

HudHealthBarOptions Reference:

Property Data type Default value Description
w number 200 Width of the health bar
h number 30 Height of the health bar
value number 100 Current value of the health bar
barColor string "0xFF0000" Color of the health bar
textColor string "0xFF0000" Color of the text displaying the value
