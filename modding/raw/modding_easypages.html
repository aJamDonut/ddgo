Easy Pages

Creates a container with multiple pages for easy navigation.

Syntax

game.render.component('easypages', EasyPagesOptions);​

Example

game.render.component('easypages', {
    pages: 5,
    w: 500,
    h: 500
});​

EasyPagesOptions Reference:

Property Data type Default value Description
pages number 5 Number of pages to create
w number 500 Width of the container
h number 500 Height of the container
