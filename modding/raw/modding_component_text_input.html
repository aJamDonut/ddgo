Text Input

Provides the ability for the user to input text through the keyboard.

Syntax

game.render.component('component_text_input', TextInputOptions);​

Example

game.render.component('component_text_input', {
  text: 'example text',
  w: 200,
  onChange: function(text) {
    console.log(text);
  }
});​

TextInputOptions Reference:

Property Data Type Default Value Description
text string "I'm here" The default value for the input area
w number 300 Width of the input area
onChange function undefined Specifies a function to run when the value of the input changes
