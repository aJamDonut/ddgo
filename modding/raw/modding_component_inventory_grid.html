Inventory Grid

Displays an inventory grid with customizable number of columns and rows.

Syntax

game.render.component('component_inventory_grid', InventoryGridOptions);​

Example

game.render.component('component_inventory_grid', {
    cols: 3,
    rows: 4,
    inventory: {
        item1: {name: 'Sword', quantity: 1},
        item2: {name: 'Shield', quantity: 2},
        item3: {name: 'Potion', quantity: 5}
    }
});​

InventoryGridOptions Reference:

Property Data type Default value Description
cols number 2 Number of columns in the grid
rows number 2 Number of rows in the grid
inventory object {} Object representing the inventory items
