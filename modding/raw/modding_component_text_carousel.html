Text Carousel

Displays a carousel of text items with navigation buttons to scroll through the items.

Syntax

game.render.component('component_text_carousel', TextCarouselOptions);​

Example

game.render.component('component_text_carousel', {
    w: 500,
    h: 500,
    items: ['Title 1', 'Title 2', 'Title 3'],
    bottomTexts: ['Here is some example text', 'Example 2', 'Example 3']
});​

TextCarouselOptions Reference:

Property Data type Default value Description
w number 500 Width of the carousel
h number 500 Height of the carousel
items array ['Title 1', 'Title 2', 'Title 3'] Array of text items to display in the carousel
bottomTexts array ['Here is some example text', 'Example 2', 'Example 3'] Array of text to display below each item in the carousel
topFill string '0x212121' Color fill for the top pane
buttonFill string '0x3d3d3d' Color fill for the navigation buttons
bottomFill string '0x2A2A2A' Color fill for the bottom pane
textFill string '0x3d3d3d' Color fill for the text in the bottom pane
