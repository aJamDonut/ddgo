List Selector

Provides a UI component for selecting items from two lists and moving them between the lists.

Syntax

game.render.component('component_list_selector', ListSelectorOptions);​

Example

game.render.component('component_list_selector', {
    w: 300,
    h: 200,
    leftList: ['left', 'two', 'three'],
    rightList: ['another1']
});​

ListSelectorOptions Reference:

Property Data type Default value Description
w number 300 Width of the list selector
h number 200 Height of the list selector
leftList array ['left', 'two', 'three'] Items to display in the left list
rightList array ['another1'] Items to display in the right list
