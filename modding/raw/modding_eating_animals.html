Eating Animals

This component allows the player to choose between different animals to eat.

Syntax

game.render.component('eating_animals', EatingAnimalsOptions);​

Example

game.render.component('eating_animals', {
    w: 450,
    h: 150
});​

EatingAnimalsOptions Reference:

Property Data type Default value Description
w number 450 Width of the component
h number 150 Height of the component
