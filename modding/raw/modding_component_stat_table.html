Stat Table

Displays a table of stats for an item, such as armor or weapon.

Syntax

game.render.component('component_stat_table', StatTableOptions);​

Example

game.render.component('component_stat_table', {
    stats: {
        'maxSpeed': 10,
        'maxHP': 100,
        'maxWeight': 50
    },
    oWidth: 500,
    showResourceIcons: true
});​

StatTableOptions Reference:

Property Data type Default value Description
stats object {} Object containing the stats to display in the table
oWidth number 450 Width of the table
showResourceIcons boolean true Whether to show resource icons for each stat
