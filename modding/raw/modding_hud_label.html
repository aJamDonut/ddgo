Button

Provides a clickable button component with customizable options.

Syntax

game.render.component('hud_button', HudButtonOptions);​

Example

game.render.component('hud_button', {
    name: 'Click Me',
    w: 200,
    h: 50,
    fill: '0xff0000',
    onClick: () => {
        console.log('Button clicked');
    }
});​

HudButtonOptions Reference:

Property Data type Default value Description
name string "NONAME" Text to display on the button
w number 150 Width of the button
h number 40 Height of the button
pass boolean false Whether to pass an object
fill string "0x3d3d3d" Color fill of the button to match the theme
onClick function undefined Function to run when the button is clicked
