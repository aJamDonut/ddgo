Radio Button

Provides the ability to select a single option from a group of options.

Syntax

game.render.component('hud_radiod_button', HudRadiodButtonOptions);​

Example

game.render.component('hud_radiod_button', {
    name: 'Option 1',
    isChecked: true,
    onClick: () => {
        console.log('Option 1 selected');
    }
});​

HudRadiodButtonOptions Reference:

Property Data type Default value Description
name string "NONAME" Text to show next to the radio button
w number 150 Width of the radio button
h number 40 Height of the radio button
pass boolean false Whether to pass an object
fill string "0x3d3d3d" Color fill of the radio button
isChecked boolean false Whether the radio button is initially checked
onClick function undefined Function to run when the radio button is clicked
