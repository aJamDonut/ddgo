Pager List Component

A component that displays a list of items in pages, allowing the user to navigate through the pages.

Syntax

game.render.component('pagerlist', PagerListOptions);​

Example

game.render.component('pagerlist', {
    items: [item1, item2, item3],
    itemsPerPage: 5,
    page: 1,
    ticks: true
});​

PagerListOptions Reference:

Property Data type Default value Description
items array [] Array of items to display
itemsPerPage number 10 Number of items to display per page
page number 1 Current page number
ticks boolean false If true, the component won't refresh when changing pages

PagerList Methods:

Method Description
refresh Refreshes the component, updating the displayed items and page buttons
pageUp Moves to the next page, if available
pageDown Moves to the previous page, if available
empty Removes all items from the component
addPageButtons Adds custom page buttons to the component
