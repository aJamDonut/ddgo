Stat Bar

Displays a bar representing a stat value, such as health points (HP).

Syntax

game.render.component('statBar', StatBarOptions);​

Example

game.render.component('statBar', {
    name: 'HP',
    value: 75,
    maxValue: 100,
    fill: 0x990000
});​

StatBarOptions Reference:

Property Data type Default value Description
w number 200 Width of the stat bar
h number 25 Height of the stat bar
name string "HP" Name of the stat
value number 0 Current value of the stat
fill number 0x990000 Color to fill the stat bar
maxValue number 100 Maximum value of the stat
padding number 5 Padding around the stat bar
