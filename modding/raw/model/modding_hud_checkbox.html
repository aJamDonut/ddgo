Checkbox

Provides the ability to toggle a feature on or off, aswell as multi choice options.

Syntax

game.render.component('hud_checkbox', HudCheckboxOptions);​

Example

game.render.component('hud_button', {
 name: 'Enable Audio',
 isEnabled: ()=>{
  game._SETTINGS['audio'];
 },
 onEnable: ()=>{
  game._SETTINGS['audio'] = true;
 },
 onDisable: ()=>{
  game._SETTINGS['audio'] = false;
 }
});​

HudCheckboxOptions Reference:

Property Data type Default value Description
name string "Disabled" | "Enabled" Text to show on the button
isEnabled function undefined Function to check if it is enabled
onEnable function undefined Function to run when enabled
onDisable function undefined Function to run when disabled
w number 150 Width of the button
h number 40 Height of the button
lblW number 150 Width of the label
