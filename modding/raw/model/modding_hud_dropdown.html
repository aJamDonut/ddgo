Dropdown

Provides a dropdown component with selectable items.

Syntax

game.render.component('hud_dropdown', HudDropdownOptions);​

Example

game.render.component('hud_dropdown', {
    name: 'Dropdown Example',
    x: 100,
    y: 200,
    w: 200,
    h: 40,
    items: ['Option 1', 'Option 2', 'Option 3'],
    selected: 'Option 2'
});​

HudDropdownOptions Reference:

Property Data type Default value Description
name string "NONAME" Name of the dropdown component
x number 0 X-coordinate of the dropdown component
y number 0 Y-coordinate of the dropdown component
w number 150 Width of the dropdown component
h number 40 Height of the dropdown component
items array ['test1', 'test2'] Array of items to display in the dropdown
selected string 'test2' Default selected item in the dropdown
