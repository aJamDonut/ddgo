Tab Container

A container that displays multiple tabs, each containing a different component.

Syntax

game.render.component('component_container_tabs', TabContainerOptions);​

Example

game.render.component('component_container_tabs', {
    tabNames: ['Tab 1', 'Tab 2', 'Tab 3'],
    tabComponents: [
        game.render.component('hud_button', {name: 'Button 1'}),
        game.render.component('hud_button', {name: 'Button 2'}),
        game.render.component('hud_button', {name: 'Button 3'})
    ],
    w: 500,
    h: 300
});​

TabContainerOptions Reference:

Property Data type Default value Description
tabNames array ['tab1', 'tab2', 'tab3', 'new', 'brt', 'cod', 'hug', 'uhd'] Array of tab names
tabComponents array [game.render.component('hud_button', {name: 'hud1'}), game.render.component('hud_button', {name: 'hud2'}), game.render.component('hud_button', {name: 'hud3'})] Array of components to display in each tab
w number 500 Width of the tab container
h number 300 Height of the tab container
