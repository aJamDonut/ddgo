Money Label

Displays a label showing the amount of money.

Syntax

game.render.component('moneyLabel', MoneyLabelOptions);​

Example

game.render.component('moneyLabel', {
    name: 'Money',
    value: 100,
    x: 10,
    y: 10,
    w: 150,
    h: 30,
    lblW: 140
});​

MoneyLabelOptions Reference:

Property Data type Default value Description
name string "NONAME" Name of the money label
value number 0 Value of the money
x number 0 X position of the money label
y number 0 Y position of the money label
w number 100 Width of the money label
h number 30 Height of the money label
lblW number 90 Width of the label pane
