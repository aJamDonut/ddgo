Button

A clickable button with customizable text and width.

Syntax

game.render.component('hud_button', HudButtonOptions);​

Example

game.render.component('hud_button', {
     name: 'Click Me',
     w: 200,
     h: 50,
     onClick: () => {
             // Action to perform on click
        }
     });​

HudButtonOptions Reference:

Property Data type Default value Description
name string "NONAME" Text to display on the button
w number 150 Width of the button
h number 40 Height of the button
pass boolean false Whether or not to pass the button instance to the onClick function
lblW number 150 Width of the text label on the button
onClick function Function to execute when the button is clicked
