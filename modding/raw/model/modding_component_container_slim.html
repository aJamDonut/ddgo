Slim Container

Creates a slim pane with optional title and close button.

Syntax

game.render.component('component_container_slim', SlimContainerOptions);​

Example

game.render.component('drawSlimPane', {
    x: 100,
    y: 100,
    w: 200,
    h: 100,
    title: 'Settings',
    showX: true,
    titleW: 180,
    titleH: 60,
    name: 'settingsPane'
});​

SlimContainerOptions Reference:

Property Data type Default value Description
x number 0 X position of the pane
y number 0 Y position of the pane
w number 64 Width of the pane
h number 64 Height of the pane
title string | false false Title text to display on the pane
showX boolean false Whether to show the close button
titleW number 180 Width of the title pane
titleH number 60 Height of the title pane
name string "NONAMEADDED" Name of the pane for identification
