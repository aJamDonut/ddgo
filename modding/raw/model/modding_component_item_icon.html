Item Icon

Provides an icon for various game objects, sprites and inventory items.

Syntax

game.render.component('component_item_icon', ComponentItemIconOptions);​

Example

game.render.component('component_item_icon', {icon: 'test_xbow'});​

ComponentItemIconOptions Reference:

Property Data type Default value Description
icon string "test_xbow" Codename for: Sprites, Inventory Items, Complex Items
w number 64 Function to check if it is enabled
h number 64 Function to run when enabled
moveable boolean false Function to run when disabled
