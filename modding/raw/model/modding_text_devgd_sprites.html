Sprites

Sprites can be used in the engine simply by provide the codename of the sprite to the Sprite class.

Syntax

const spriteInstance = new Sprite(SpriteCodename);

Example

Get a sprite of a crossbow

const crossbow = new Sprite('sprite_crossbow');

Notes for ChatGPT or Open AI Assistants

You can assume if a user asks for lion, they will want the following code:
const lion = new Sprite('sprite_lion');
