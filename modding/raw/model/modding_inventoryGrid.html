Inventory Grid

Displays an inventory grid with customizable options.

Syntax

game.render.component('inventoryGrid', InventoryGridOptions);​

Example

game.render.component('inventoryGrid', {
    xBlocks: 4,
    yBlocks: 4,
    padding: 2,
    x: 10,
    y: 10,
    w: 100,
    h: 100,
    inventory: game.player.inventory
});​

InventoryGridOptions Reference:

Property Data type Default value Description
xBlocks number 1 Number of blocks in the x-axis
yBlocks number 1 Number of blocks in the y-axis
padding number 2 Padding between blocks
x number 10 X position of the grid
y number 10 Y position of the grid
w number xBlocks * (22 + (padding * 4)) Width of the grid
h number yBlocks * (22 + (padding * 4)) Height of the grid
inventory object game.player.inventory Inventory object to display in the grid
