drawBoxPane

Creates a customizable pane with a title bar, moveable and close features.

Syntax

game.render.component('drawBoxPane', drawBoxPaneOptions);​

Example

game.render.component('drawBoxPane', {
      x: 100,
      y: 100,
      w: 200,
      h: 200,
      title: "Settings",
      showX: true,
      onClose: () => {
           console.log('pane closed');
      },
      moveable: true,
      onTop: true
});​

drawBoxPaneOptions Reference:

Property Data type Default value Description
x number 0 X position of the pane
y number 0 Y position of the pane
w number 500 Width of the pane
h number 500 Height of the pane
title string|false false Text to be added to the title bar
showX boolean false Whether or not to show a close button
titleW number 180 Width of the title bar
titleH number 35 Height of the title bar
name string "NONAMEADDED" Name of the pane
callerObject object|false false The object that called the pane
onClose function|false false Function to run when pane is closed
moveable boolean false Whether or not the pane is moveable
onTop boolean false Brings the pane to the top when clicked on
bg string|false 'sprite_texture_scuff' Image to use as background
