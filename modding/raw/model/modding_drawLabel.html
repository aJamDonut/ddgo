Label

A label: value general purpose for displaying stats.

Syntax

const label = game.render.component('drawLabel', HudLabelOptions);​

Example

game.render.component('drawLabel', {
 name: 'Money',
 value: 100
});​

HudLabelOptions Reference:

Property Data type Default value Description
name string NONAME Text to show for the label
value string | number 0 Text to show for the value
w number 250 Width of the label
h number 30 Height of the label
