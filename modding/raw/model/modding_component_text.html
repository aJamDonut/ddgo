Text

Display text on the page, must use pre-defined styles.

Syntax

game.render.component('component_text', ComponentTextOptions);​

Example

game.render.component('component_text', {text: 'Hello world!', style: 'info-tagline'});​

ComponentTextOptions Reference:

Property Data type Default value Description
text string "Hello world!" The text to display
style string "info-tagline" The style name to show the text in
