Counter

Provides a numerical counter with adjustable increments and a maximum and minimum value. Includes buttons to increase and decrease the count.

Syntax

game.render.component('hud_counter', CounterOptions);​

Example

game.render.component('hud_counter', {
  ref: {exampleCount: 0},
  value: 'exampleCount',
  min: 0,
  max: 10,
  multi: 2,
  w: 200
});​

CounterOptions Reference:

Property Data type Description Default value
ref object The object that contains the count value. undefined
value string The key of the count value in the ref object. undefined
min number The minimum value the count can be. 0
max number The maximum value the count can be. 10
multi number The amount by which the count changes with each button press. 1
w number The width of the counter element. 100
